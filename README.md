# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, developed using **HTML5** and **CSS3**. The site presents a structured overview of my professional background, technical skills, projects, education, and contact information.

## 📁 Project Structure
The website consists of multiple well-organized pages:
- **Home** – Introduction and professional summary.  
- **About** – Background, certifications, and achievements.  
- **Projects** – Key academic and technical projects.  
- **Skills** – Technical and core computer science competencies.  
- **Education** – Academic qualifications presented in a timeline format.  
- **Contact** – Contact details and a message section.

## 🛠 Technologies Used
- **HTML5** for page structure  
- **CSS3** for styling, layout, gradients, and responsive design elements  
- **Font Awesome** for icons
